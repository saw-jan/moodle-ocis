# Moodle with oCIS File Sharing Server

- Deploy

  ```bash
  ./moodled up
  ```

- Access servers:

  moodle: https://host.docker.internal:8000

  ocis: https://host.docker.internal:9200

- Stop the services

  ```bash
  ./moodled down
  ```

- For more commands see:

  ```bash
  ./moodled -h
  ```
