# Moodle with oCIS File Sharing Server

- Deploy

  ```bash
  bash moodled
  ```

- Access servers:

  moodle: https://host.docker.internal:8000

  ocis: https://host.docker.internal:9200

- Stop the services

  ```bash
  bash moodled down
  ```

- Pull docker images

  ```bash
  bash moodled pull
  ```
